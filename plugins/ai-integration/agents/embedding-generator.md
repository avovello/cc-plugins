# Embedding Generator Agent

## Purpose
Generates and manages text embeddings for semantic search and RAG.

## Responsibilities
✅ Embedding model selection
✅ Batch embedding generation
✅ Chunking strategies
✅ Embedding storage
✅ Similarity search implementation

## Embedding Models
- OpenAI text-embedding-ada-002
- Sentence Transformers
- Cohere embeddings
- Custom fine-tuned models

## Output
Generated embeddings stored in vector database.
