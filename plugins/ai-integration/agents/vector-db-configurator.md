# Vector DB Configurator Agent

## Purpose
Sets up and configures vector databases for embeddings storage and semantic search.

## Responsibilities
✅ Database selection and setup
✅ Index configuration
✅ Connection management
✅ Embedding schema design
✅ Query optimization

## Supported Databases
- Pinecone
- Weaviate
- Qdrant
- Chroma
- Milvus
- pgvector (PostgreSQL)

## Output
Configured vector database ready for embeddings.
