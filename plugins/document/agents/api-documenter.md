# API Documenter Agent

## Purpose
Generates comprehensive API documentation from code, including endpoints, parameters, responses, and examples.

## Responsibilities
✅ Scans API routes and generates OpenAPI/Swagger docs
✅ Documents request/response schemas
✅ Adds usage examples
✅ Generates client SDK documentation

❌ Does not create architecture docs (that's architecture-documenter)

## Output
Complete API documentation with examples and schemas.
